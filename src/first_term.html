Discrete Probability Distributions

Variables that take countable outcomes with probabilities that add up to 1.

Core Distributions

  • Bernoulli(p): Single trial with success probability p; mean p, variance p(1-p).
  • Binomial(n, p): Number of successes in n Bernoulli trials; PMF C(n, k)p^k(1-p)^{n-k}.
  • Geometric(p): Trials until first success; PMF (1-p)^{k-1}p; mean 1/p.
  • Poisson(λ): Counts of rare independent events in a fixed interval; PMF e^{-λ}λ^k / k!; mean=variance=λ.

Expectation & Variance

For a discrete random variable X with PMF f(x), expectation is E[X] = Σ x·f(x) and variance is Var(X) = E[X^2] - (E[X])^2.

Example: For a fair die, E[X] = (1+2+3+4+5+6)/6 = 3.5 and Var(X) = E[X^2] - (E[X])^2 = (1^2+...+6^2)/6 − 3.5^2 ≈ 2.92.

Common Uses

  • Quality control (binomial defects per batch).
  • Modeling arrivals or errors over time (Poisson counts).
  • Number of trials needed for a success (geometric).
